Fitribe X 新營養食代-營養師帶你瘦身

Fitribe X 新營養食代 七月中的時候抽空拜訪了台中的營養師團隊，從一開始做 Fitribe 這個題目我們就想要接觸營養師，這算是第一次正式的找到營養師的團隊。在台中的時候算是相談甚歡吧！感覺得出來他們對 App 平台有滿大的興趣，也隱約感受到目前的營運並不是很好，雖然他們推出了 Web的飲食記錄服務，但是一般消費者會直接來使用的比例非常的低，其實都是靠著背後中國醫藥大學的老師靠著人脈關係帶進來的使用者。實際上離平台化還有非常遠的一段距離。 在去之前我們就想了要提出營養師線上諮詢的試營運服務，當下兩邊也很有意願想要嘗試。回來以後我跟竹田如火如荼的展開準備，一邊修改程式，一邊準備著試營運要用的廣告素材。上週六我們又去了南港的生技展找他們討論試營運的細節，這次的體驗就不是這麼好了。言談之中覺得他們是想利用 Fitribe 這個平台來販賣商品，給用戶營養師的服務反而是其次，也和我們當初想透過販售營養師服務來盈利的目標相距甚遠。 我不知道這樣子是好是壞，的確有可能他們是根據過往的經驗再給我們建議，但是都透過他們來介紹廠商，反而讓我有不切實的感覺。而且幾乎都是和中國醫藥大學的教授和他的夥伴在談，實際上並不是在接觸第一線的營養師，總有點隔山打牛的意味與辛苦。 Well，我們還是很努力的在準備這次試營運的活動，希望會有好的結果囉… 至少這次我們很努力的嘗試，不論結果好壞，提醒自己過程是最重要的，對吧！
